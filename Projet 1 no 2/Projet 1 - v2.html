Code

Tout votre code doit tre crit la main et vos pages web doivent tre statiques.

Donc, vous ne pouvez pas utiliser:

HTML & CSS

3 pages HTML avec au moins un fichier CSS.

Essayez d'organiser votre CSS pour qu'il n'y ai pas de duplication de code. Des duplications de code excessive entraineront des pertes de points. Tout votre style devrait tre contenu dans des fichiers CSS et non dans des attributs/balises style mme votre fichier HTML

Vous tes encourags :

  • Ne pas inclure des balises SVG directement dans le HTML, plutt relier un fichier .svg (dans un cas pour lequel il y a un SVG dans la page)
